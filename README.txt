How to make it work in Windows

1) Install node.js: https://nodejs.org/en/#download
2) Install Git Bash: https://git-scm.com/downloads
3) Open Git Bash as administrator or use 'sudo' command.
4) Go to the location of the project, open its main folder.
5) Execute: npm install --save express
6) Run it: node server.js
7) This will run the application in a localhost server, showing something like: listening on *:3000
8) Open the browser and go to: localhost:3000

Possible problems in Windows - "npm" command is not recognised: http://www.hacksparrow.com/install-node-js-and-npm-on-windows.html
